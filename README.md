# Auto PR Review Bot (Free)

This repository provides a **fully free**, offline, rule-based GitHub Actions bot
that automatically reviews pull requests.

## Features
- Automated PR comments
- Detects TODOs
- Detects large functions
- Detects missing docstrings
- Detects debug prints
- Fully free (no API keys, no paid AI)

## How to use
1. Fork or clone this repo
2. Commit your code
3. Push to GitHub
4. Open a pull request
5. GitHub Actions auto-comments review
